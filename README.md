# SomaTime 기획서 통합 레포

SW마에스트로 AI 기술교육 31조의 프로젝트 기획서를 모으고 통합하기 위한 문서 레포입니다.

현재 단계에서는 구현 코드를 두지 않습니다. 5명의 팀원이 각자 작성한 기획서를 `기획서/` 아래에 올리고, 2026-05-01 회의에서 하나의 방향으로 통합합니다.

## 현재 목표

- 2026-05-01: 팀원 5명의 개인 기획서 초안 공유
- 2026-05-02 23:59: 프로젝트 기획서 제출
- 2026-05-10 23:59: 데모 코드 제출
- 2026-05-15 13:30-15:30: 최종 발표/데모데이

## 파일 구조

```text
.
├── README.md
├── 전체일정.md
├── 작성가이드.md
└── 기획서/
    ├── 공통_기획서_양식.md
    ├── park-sejong.md
    ├── ahn-soobin.md
    ├── lee-joonggon.md
    ├── ji-sanggeun.md
    ├── hong-jiyeon.md
    ├── 통합본.md
    └── 제출본.md
```

## 작성 방식

1. 각자 자기 이름의 파일을 수정합니다.
2. 5월 1일 회의 전까지 3분 설명이 가능할 정도로 채웁니다.
3. 회의에서 공통 방향을 정한 뒤 `기획서/통합본.md`에 합칩니다.
4. 5월 2일 제출 직전 `기획서/제출본.md`를 최종본으로 고정합니다.

원본 양식 설명은 `기획서/공통_기획서_양식.md`에 보관되어 있으며, 각 팀원별 템플릿에도 같은 설명을 포함했습니다.

## 현재 가설

SomaTime은 개인 일정 세부 내용을 공개하지 않고도 여러 사람의 공통 가능 시간을 빠르게 찾는 개인정보 보호 일정 조율 서비스입니다.

핵심 원칙:

- 원본 캘린더를 복제하지 않습니다.
- 일정 제목, 장소, 설명을 저장하지 않습니다.
- 회의 조율에 필요한 busy/free availability block만 사용합니다.
- 많은 기능보다 하나의 안정적인 일정 조율 흐름에 집중합니다.
