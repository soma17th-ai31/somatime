## 6. 현재 프로젝트 방향 초안

- 프로젝트명: SomaTime(가칭)
- 주제: 개인정보 보호가 되는 공유캘린더
- 핵심 문제: 여러 사람이 일정을 맞출 때 개인 일정 세부 내용은 보호하면서 공통 가능 시간을 빠르게 찾기 어렵다.
- 대상 사용자: SW마에스트로 연수생, 팀원, 멘토, 엑스퍼트, 코치
- 대표 시나리오: 팀원 3명과 멘토 1명의 1시간 멘토링 시간을 잡는 상황
- MVP 기준: 개인 일정 세부 내용을 공개하지 않고 공통 후보 시간 3개를 안정적으로 추천할 수 있으면 성공

## 7. MVP 범위 초안

포함 후보:

- 새 회의 생성
- 참여자별 가능 시간 입력
- busy/free/preferred block 정리
- 공통 가능 시간 계산
- 후보 시간 3개 추천
- 추천 이유와 복사용 공유 메시지 생성
- 로컬 실행 가능한 데모와 README

제외 후보:

- 외부 캘린더 OAuth 실시간 연동
- 로그인/회원가입
- 모바일 앱
- 실시간 알림
- 자동 메시지 발송
- 자동 일정 등록
- 여러 회의 동시 최적화
- 원본 일정 제목, 장소, 설명 저장

추가 검토:

- `.ics` import를 넣을지
- CSV/텍스트 paste를 넣을지
- 직접 입력 중심으로 단순화할지
